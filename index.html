GIT - Intro

Git merupakan salah satu version control system yang diciptakan oleh Linus Torvalds yang berguna untuk mencatat setiap perubahan file pada code atau sourcecode
Git memiliki manfaat yaitu:

  • Menyimpan seluruh versi sourcecode
  • Memudahkan untuk berkolaborasi saat membangun sebuah sistem
  • Dapat memahami deploy modern
  • Dll.

    • Adapun tools yang digunakan git yaitu:
  • Terminal
  • Github/Gitlab
  • Sourcetree
  • Texteditor
  • Dll.

    • GIT - Fundamental

    Perintah dasar yang sering digunakan oleh git disebut Fundamental Git. Salah satu manfaat dari fundamental git adalah mempercepat dalam pengerjaan sebuah project.
    Adapun macam-macam fundamental git yaitu:

  • Git Init - berguna untuk menginisialisasikan sebuah repository
  • Git Config - berguna untuk configurasi email dan username baik config local maupun global
  • Git Status - berguna untuk mengetahui ststus file
  • Git Add . - berguna untuk memasukan file ke staging area
  • Git Commit - berguna untuk mencatat progres atau update
  • Git Log - berguna untuk melihat riwayat commit
  • Git Clone - berguna untuk cloning atau copy repository dari server ke local
  • Git Push - berguna untuk mengirim commit kepada server
  • Git Remote - berguna untuk berinteraksi antara client dengan server
  • Git Pull - berguna untuk mengambil commit dari server jika ada perubahan di server yang dibuat oleh branch lain

    • GIT - Branch

    Branch atau bisa disebut juga dengan cabang atau copy dari repository. Branch biasanya digunakan untuk memanajemen pengembangan.
    Adapun macam-macam branch yaitu:

  • git brach - berguna untuk melihat semua branch pada local
  • git brach --all - berguna untuk melihat semua branch pada public
  • git brach nama_brach - berguna untuk menambahkan/membuat brach baru
  • git checkout nama_branch - berguna untuk mengganti/berpindah branch
  • git branch -d nama_brach - berguna untuk menghapus branch
  • git merge nama_brach - berguna untuk menggabungkan branch

    • GIT - GUI

    Salah satu software berbasis GUI untuk memudahkan mengoperasikan git yaitu Sourcetree.
    Sourcetree digunakan untuk memudahkan dalam solving conflict dan mempercepat kinerja. Untuk mensolving conflict menggunakan sourcetree perlu mendownload diffmerge terlebih dahulu dan dikonfigurasi melalui tool-options.
